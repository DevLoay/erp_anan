# Settings & Permissions Assignment Guide

تم تحسين شاشة إدارة المستخدمين والصلاحيات لتكون مبنية على دور + نطاق بيانات واضح:

- ADMIN: صلاحية كاملة ولا يحتاج نطاق.
- OPERATION_MANAGER: تشغيل عام مع صلاحيات واسعة بدون إعدادات حساسة.
- SUPERVISOR: يفضل ربطه بمشرف أو مدينة.
- ACCOUNTANT: الماليات والمسير، ويفضل تحديد المدينة/المشروع.
- HR: المناديب والموارد البشرية، ويفضل تحديد المدينة.
- VIEWER: عرض فقط.

قبل حفظ أي مستخدم، تظهر بطاقة معاينة للصلاحية والنطاق والتنبيهات.
